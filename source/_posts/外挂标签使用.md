> 内置的外挂标签满足大部分用户的需求，不需要额外安装插件。
> 插件的外挂标签需要安装对应的插件，满足部分用户的特殊需求。

# 内置标签

## Chart

图表标签，用于显示图表。
打开配置文件中开关

```yaml
chart: true
```

{% tabs Chart %}

<!-- tab 语法 -->

```markdown
{% chart %}
```

<!-- endtab -->

<!-- tab 参数 -->

不需要参数。

<!-- endtab -->

<!-- tab 示例 -->

```markdown
{% chart %}
{% endchart %}
```

<!-- endtab -->

<!-- tab 效果 -->

{% chart %}
type: 'bar',
data: {
labels: ['Red', 'Blue', 'Yellow', 'Green', 'Purple', 'Orange'],
datasets: [{
label: '# of Votes',
data: [12, 19, 3, 5, 2, 3],
borderWidth: 1
}]
},
options: {
scales: {
y: {
beginAtZero: true,
}
}
}
{% endchart %}

<!-- endtab -->

## GalleryGroup

图片组标签，用于显示图片组。

{% tabs GalleryGroup %}

<!-- tab 语法 -->

```markdown
{% galleryGroup 'name' 'desc' 'url' 'img' %}
```

<!-- endtab -->

<!-- tab 参数 -->

| 参数 | 说明         | 默认值 |
| ---- | ------------ | ------ |
| name | 图片组名称   | 无     |
| desc | 图片组描述   | 无     |
| url  | 图片组链接   | 无     |
| img  | 图片组预览图 | 无     |

<!-- endtab -->

<!-- tab 示例 -->

```markdown
<div class='gallery-group-main'>
{% galleryGroup '旅行' '出行拍摄的杂七杂八' 'https://solitude.js.org' 'https://s3.qjqq.cn/47/674c6e226855e.png!color' %}
```
